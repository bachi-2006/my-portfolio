# 🌟 About This Portfolio

Welcome to my personal portfolio website! This is a modern, interactive portfolio built to showcase my skills, projects, achievements, and journey as a Computer Science undergraduate. The site features a sleek dark theme with smooth animations and a bento-grid layout.

## 🚀 Key Features

- **🏠 Home Page**
  - Hero section with animated text (Analyze, Code & Innovate)
  - GitHub contribution streak integration
  - Interactive Like Button with animation
  - Social media links (LinkedIn, GitHub, Instagram, Linktree)
  - Resume section with download option
  - Skills showcase with tech stack icons
  - Random quotes carousel
  - Featured projects preview
  - Library section (tech resources)
  - Showcase section

- **📄 Pages**
  - **Home** - Main landing page with all features
  - **About** - Detailed about me section
  - **Portfolio** - Full projects gallery with detail sheets
  - **Blog** - Blog posts section
  - **Contact** - Contact information and form
  - **Experiments** - Terminal component and creative experiments

- **✨ Interactive Features**
  - Smooth page transitions
  - Typing and scramble text effects
  - Image zoom and rotation effects
  - Quote carousel
  - Like button functionality
  - Orbiting images animation
  - Image marquee
  - Portfolio sheet/modal for project details

## 🛠️ Tech Stack

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bachi-2006/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to:
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
